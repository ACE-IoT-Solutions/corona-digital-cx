# 1.0 Purpose and Scope

This document specifies the requirements for the digital representation and network infrastructure of the Building Automation System (BAS) as part of the digital commissioning (DCx) process. The primary goal is to ensure that all data points, equipment, system relationships, and network configurations are structured, consistent, and machine-readable from project inception. This enables advanced applications such as automated fault detection and diagnostics (AFDD), performance analytics, and seamless, truly interoperable integration with other building systems.

This specification applies to all control systems and devices associated with the mechanical (HVAC) and plumbing systems. The validation process is structured into distinct sections covering BACnet Network Infrastructure and Levels of Metadata Validation. Successful completion of all requirements is required for system acceptance.
