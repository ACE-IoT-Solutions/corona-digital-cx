# 4.2 Level 2: Tagging and Labeling Validation

## Objective

To apply a standardized, dictionary-based set of tags to all points and equipment to create a rich, queryable dataset.

## Requirements

All BAS points and equipment must be tagged using the Project Haystack 4 standard. A minimum set of tags is required for all components (e.g., `site`, `equip`, `point`, `sensor`, `cmd`, `sp`, `air`, `temp`).

## Validation and Acceptance Criteria

The submitted data model (e.g., in Haystack JSON format) must be successfully parsed and pass all predefined validation queries designed to verify tag completeness and correctness.
